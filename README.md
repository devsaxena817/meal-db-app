# 🍽️ Chef's Choice: Minimalist Meal Finder

A sleek, responsive web application that queries **TheMealDB API** to help users find the most efficient recipes. The app specifically identifies and displays the meal that requires the **least number of ingredients** from the search results, perfect for quick cooking with minimal grocery runs.

!

## ✨ Key Features
* **Smart Filter:** Automatically calculates ingredient counts across API results to find the simplest recipe.
* **Modern UI:** Features a Glassmorphism design with responsive CSS Grid layouts.
* **Real-time Feedback:** Includes a custom CSS loading spinner and error handling for empty searches.
* **Direct Video Links:** Connects users directly to YouTube cooking tutorials for the selected meal.
* **Mobile Optimized:** Fully responsive design that works on desktops, tablets, and phones.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+)
* **API:** [TheMealDB](https://www.themealdb.com/)
* **Deployment:** Netlify
* **Version Control:** Git & GitHub
